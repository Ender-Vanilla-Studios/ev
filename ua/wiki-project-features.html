Як отримати найкраще занурення в гру?

На сервері є дата паки та плагіни, вони сильно покращують геймплей.
Рекомендується використовувати Minecraft Java Edition.

Список особливостей

  • Brewery
    Плагін на пивоваріння Гайд
  • Волейбол
    Гайд на створення м'яча та його використання
  • Plasmo voice
    Голосовий чат прямо в грі
  • Whitelist ultra
    Самописний плагін
  • Кросплатформеність
    На жаль, на Bedrock виданні гри реалізовано мало контенту
  • Міст між Discord та чатом сервера
  • PlayTime
    Додаткова інформація /playtime help
  • Система скинів
    Для встановлення скину використовуйте /skin set ліцензійний-нік або по URL
  • ViaVersion
    Підтримка версій клієнта новіших за версію сервера
  • Gsit
    Плагін додає можливість сидіти і не тільки
  • Оновлення діамантових речей до незеритових без шаблону
  • Tamable Foxes
    Плагін на приручення лисиць
  • Могили (після смерті гравця)
    Вони не дадуть ресурсам пропасти
  • Наручники
    Наручники, які можна одягнути на гравців
  • End 2.0
    Перероблений Енд з новими структурами та покращеною атмосферою
  • Кастомні крафти
    Вони значно покращують сервер
  • Наші правила
    Вони дають більше свободи, ніж на 90% інших ванілл
  • Нові структури
    Вони чудово вписуються в ваніллу, фото
