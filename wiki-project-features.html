На сервере есть дата паки и плагины, они сильно улучшают геймплэй

Как получить лучшее погружение в игру?

Рекомендуется использовать Minecraft Java Edition

Список особенностей

  • Brewery
    Плагин на пивоварение Гайд
  • Воллейбол
    Гайд на создание меча и на его использование
  • Plasmo voice
    Голосовой чат прямо в игре
  • Whitelist ultra
    Самописный плагин
  • Кроссплатформенность
    К большому сожалению на Bedrock издание игры реализованно мало контента
  • Карта сервера
    В 3D
  • Мост между Discord и чатом сервера
  • PlayTime
    Доп информация /playtime help
  • Своя система скинов
    Для установки скина используете /skin set лицензионный-ник или по URl
  • ViaVersion
    Поддержка версий клиента новее версии сервера
  • Gsit
    Плагин добавляет возможность сидеть и не только
  • Обновление алмазных вещей до незеритовых без шаблона
  • Tamable Foxes
  • Магилы(полсе смерти игрока)
  • Наручники
